## Getting Started

### View Online

This portfolio is hosted on GitHub Pages:

```
https://daoninhthai.github.io/
```

Navigate to individual lessons by appending the folder name:
```
https://daoninhthai.github.io/Buoi1/b1.html
https://daoninhthai.github.io/Buoi5/animation/
https://daoninhthai.github.io/Buoi_25/index.html
```

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/daoninhthai/daoninhthai.github.io.git
   cd daoninhthai.github.io
   ```

2. **Open with a local server**

   Using Python:
   ```bash
   python -m http.server 8000
   ```

   Using Node.js:
   ```bash
   npx serve .
   ```

   Using VS Code Live Server extension: right-click any `.html` file and select "Open with Live Server."

3. **Browse the lessons**

   Open `http://localhost:8000` and navigate to any lesson directory.

### For Node.js Lessons (Buoi 7)

```bash
cd Buoi7
npm install
node index.js
```

## Learning Path Highlights

### Phase 1: Foundations (Buoi 1-6)
Core web technologies including HTML structure, CSS styling, layout techniques, animations, and responsive design principles.

### Phase 2: JavaScript & Server (Buoi 7-13)
Introduction to programming with JavaScript, DOM manipulation, event handling, and server-side development with Node.js.

### Phase 3: Advanced Topics (Buoi 15-20)
Deep dive into JavaScript patterns, API consumption, JSON data handling, and component-based architecture.

### Phase 4: Real-World Projects (Buoi 21-25)
Practical application through portfolio building, UI cloning, and enterprise-level page replication.
